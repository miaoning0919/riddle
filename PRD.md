# 知识图谱三层结构 PRD 文档

## 项目概述
将原有的 7 个节点知识图谱改造成三层分类结构（宏观层、中观层、微观层），支持多维度数据浏览。顶部只保留 3 个层级节点。

---

## 需求详情

### 一级知识图谱（顶层）

#### 结构
- **节点数量**：3 个
- **节点名称**：
  1. 宏观层
  2. 中观层
  3. 微观层

#### 连线
- **注意**：顶层**没有任何连线**（无点与点之间的连线，也无连向中心的线）
- 三个点独立分散展现

### 二层内部的连线规则

#### 连线特性
1. **不连接中心**
   - 二层内部的点不向中央大圆连线
   - 仅点与点之间按顺序连接

2. **折线连接方式**
   - 按顺序连接：第1个点 → 第2个点 → 第3个点 → ... → 最后一个点
   - 形成一条连贯的折线路径

3. **点位分布**
   - 点分散开，**非整齐的对角线排列**
   - 呈现**不太规则、有随机感的曲折分布**
   - 连线不相交

4. **视觉效果**
   - 点的位置：随机分散但有一定规律感
   - 连线：弯曲折线，呈现有机的、非对称的状态

## 交互流程

1. **初始状态**：展示中央大圆 + 三层节点已展开
2. **点击任一层**：
   - 隐藏中央大圆
   - 对应层展开为该层的多个二级节点
   - 显示该层内部的折线连接
3. **点击二级节点**：打开详情面板（保留原有逻辑）
4. **返回**：返回到上一层（保留原有返回流程）

## 技术实现要点

- 修改 `positionMenuNodes()` 实现不规则点位分布

- 修改连线生成逻辑：顶层无连线，二层按顺序择折线连接
- 保留原有的动画、交互、返回逻辑
